# Car Price Prediction App

This project is a machine learning web application built with Python and Streamlit that predicts the selling price of a used car based on user inputs.

## Features

- Predict car selling price
- User-friendly Streamlit interface
- Linear Regression model
- Sidebar input controls

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Pickle

## Input Features

- Year
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission

## Project Structure

```
car-price-prediction/
│
├── car_price_prediction_app.py
├── linear_regression_model.pkl
├── requirements.txt
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/Ailuro1234/car-price-prediction.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run car_price_prediction_app.py
```

## Example Output

The application predicts the selling price of a car based on the selected features and displays the estimated price in lakh.
